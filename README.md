# GMMW – Workshop Management System

GMMW is a full-stack workshop management system built in ASP.NET Blazor as part of a university software engineering project.

The application was designed to replace manual record-keeping with a structured digital system for managing motorists, vehicles, repairs, workshop classes, attendance, reporting, and admin functions.

## Key Features

- Motorist management
- Vehicle management
- Repair booking and repair logging
- Parts tracking and cost calculation
- Workshop class scheduling
- Attendance recording
- Monthly reporting
- Role-based admin features with ASP.NET Identity

## Tech Stack

- C#
- ASP.NET Blazor
- Entity Framework Core
- SQL
- ASP.NET Identity

## Architecture Highlights

- Multi-module business application structure
- Service-based business logic
- Entity Framework Core with relational data modelling
- Role-based access control using ASP.NET Identity
- Reporting features across workshop activity, repairs, parts, classes, and attendance
- Consistent UI design across multiple modules

## Project Structure

- `GMMW.Web` - main web application
- `GMMW.Tests` - test project

## Purpose

This project was developed as an academic university project to practise full-stack development, business rules, validation, reporting, role-based access, and consistent UI design across a larger application.

## Status

This is an active student project and is still being refined and improved.
