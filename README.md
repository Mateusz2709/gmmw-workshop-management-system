\# GMMW – Workshop Management System



GMMW is a full-stack workshop management system built in ASP.NET Blazor as part of a university software engineering project.



The purpose of the application is to replace manual record-keeping with a structured digital system for managing motorists, vehicles, repairs, workshop classes, attendance, reporting, and admin functions.



\## Features



\- Motorist management

\- Vehicle management

\- Repair booking and repair logging

\- Parts tracking and cost calculation

\- Workshop class scheduling

\- Attendance recording

\- Reporting

\- Role-based admin features with ASP.NET Identity



\## Tech Stack



\- C#

\- ASP.NET Blazor

\- Entity Framework Core

\- SQL

\- ASP.NET Identity



\## Project Structure



\- `GMMW.Web` – main web application

\- `GMMW.Tests` – test project



\## Purpose



This project was developed as an academic university project to practise full-stack application development, business logic, validation, reporting, role-based access, and consistent UI design across a larger multi-module system.



\## Status



This is an active student project and is still being refined and improved.
